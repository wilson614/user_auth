## 環境 Environment
- Node.js
- Nodemon
- mongodb

## 安裝 Installation

1. 取得專案至本地
```
git clone https://github.com/wilson614/user_auth
```
2. 進入專案目錄
```
cd user_auth
```
3. 安裝套件
```
npm install
```
4. 產生種子資料
```
npm run seed
```
5. 啟動專案
```
npm run dev
```
6. 出現以下訊息後，即可在 http://localhost:3000/login 開始使用
```
This server is running on http://localhost:3000/login
```
